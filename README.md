# Cloud Agent Handoff Kit

A ready-to-fill handoff packet for delegating one repo task to a cloud coding agent and accepting only evidence-backed PRs.

## Why this exists
Inspired by: https://x.com/cursor_ai/status/2067366343817805899

## What it does
- Provides a first-trial packet for one real workflow.
- Includes an infographic explaining the operating model.
- Keeps a draft skill separate until evidence proves reuse.

## Quickstart
Open prototype/Trial Operator Handoff Checklist.md first, then fill prototype/Cloud Agent Handoff Packet.md during a real repo task, complete prototype/Cloud Agent Promotion Decision Card.md, and finish prototype/Cloud Handoff Post-Trial Debrief Template.md before changing claims.
