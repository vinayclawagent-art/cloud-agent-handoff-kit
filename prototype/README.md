# Cloud Agent Handoff Kit Prototype

Source: [[Cursor Cloud Subagents From Phone to PR]]

Latest improvement: 2026-06-21 added [[Cloud Agent Handoff Kit/Claim-to-Patch Ledger]] so future evidence, rubric notes, and promotion decisions become exact README/prototype/skill patch targets before wording changes; evidence still pending.

## What it demonstrates
A ready-to-fill handoff packet for delegating one repo task to a cloud coding agent and accepting only evidence-backed PRs.

## How to use
1. Open [[Cloud Agent Handoff Kit/Trial Operator Handoff Checklist]] and assign the next real trial operator.
2. Open [[Cloud Agent Handoff Kit/Cloud Agent Handoff Packet]].
3. Fill it during one real task.
4. Attach concrete proof links before changing claims in the package README or skill draft.
5. Fill [[Cloud Agent Handoff Kit/Trial Evidence Index]] to map each proof item to the claim it supports.
6. Grade the proof with [[Cloud Agent Handoff Kit/Cloud Handoff Evidence Quality Rubric]].
7. Fill [[Cloud Agent Handoff Kit/Claim-to-Patch Ledger]] to choose the exact claims and files that are allowed to change.
8. Complete [[Cloud Agent Handoff Kit/Cloud Agent Promotion Decision Card]] to decide promote / pilot-only / iterate / hold.
9. Fill [[Cloud Agent Handoff Kit/Cloud Handoff Post-Trial Debrief Template]] before patching public claims.
